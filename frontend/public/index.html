リアルタイム感情分析

SNSの投稿を自動収集し、AWS Bedrockによる高精度な感情分析でブランドの評判をリアルタイムで監視

アプリを起動

主要機能

自動投稿収集

Twitter、Instagram、FacebookなどのSNSから関連投稿を自動収集。キーワードベースの柔軟な検索設定が可能。

AI感情分析

AWS Bedrock Novaを活用した高精度な感情分析。ポジティブ、ネガティブ、ニュートラルを詳細にスコアリング。

リアルタイム可視化

Gradioベースの直感的なダッシュボードで、感情の変化やトレンドをリアルタイムで可視化。

アラート機能

異常なネガティブスパイクや急激な感情変化を検知し、即座にアラートでお知らせ。

データエクスポート

分析結果をCSV、JSON形式でエクスポート。外部システムとの連携も簡単に実現。

セキュアな基盤

AWS CloudFront、API Gateway、Lambdaを使用した安全でスケーラブルなインフラ構成。
